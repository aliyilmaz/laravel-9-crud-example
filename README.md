## Laravel CRUD Example

Example of create, update, delete and read operations in laravel 9 version.

### Download

Step 1
```php
git clone https://github.com/aliyilmaz/laravel-crud-example.git
```
Step 2
```php
composer install
```

### Install
```php
php artisan migrate
```

### Run
```php
php artisan serve
```
